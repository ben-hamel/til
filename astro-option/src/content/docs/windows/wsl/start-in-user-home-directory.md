---
title: Start WSL in User Home Directory
description: Use wsl ~ to launch WSL directly in your home directory from any Windows prompt.
---

By default, WSL opens in the current Windows directory. Passing `~` as an argument starts WSL directly in the Linux user's home directory instead.

## Usage

```bash
wsl ~
```

**Source:** [Microsoft Docs - Basic WSL Commands](https://learn.microsoft.com/en-us/windows/wsl/basic-commands#start-wsl-in-user-home-directory)
